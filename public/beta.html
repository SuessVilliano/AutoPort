📖 How to Use AutoPort

Step-by-step guide for porting phone numbers into GoHighLevel

1

Check Portability

Enter the phone number(s) on the main form. AutoPort calls the Twilio Portability API to verify each number can be ported. You'll see the carrier name, number type (local/mobile/toll-free), and whether it's portable.

2

Submit Port Request

Fill in the customer's carrier account details — account name, account number, PIN, and service address. AutoPort uploads the required documents and creates a Port-In Request via Twilio's API.

3

Sign the LOA

The customer receives an email with a link to sign the Letter of Authorization (LOA). This is required by carriers to authorize the port. Once signed, the port is submitted to the losing carrier.

4

Track Status via Webhooks

Twilio sends real-time webhook updates as the port progresses. AutoPort automatically sends branded emails to the customer at each stage: submitted, FOC date confirmed, completed, or rejected.

5

Handle Rejections

If a port is rejected (common — usually an account info mismatch), the customer gets an email with a "Fix & Resubmit" link. They correct the info and AutoPort resubmits automatically.
